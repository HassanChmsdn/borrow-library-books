Feature-oriented modules live in this directory.

- `catalog/` holds the reader-facing browse experience.
- `borrowings/` holds the member-facing borrowing views.
- `profile/` holds the member account experience.
- `admin-dashboard/` holds the library operations dashboard.
- `admin-books/` holds catalog management surfaces for admins.
- `admin-categories/` holds collection category planning surfaces.
- `admin-borrowings/` holds circulation oversight workflows.
- `admin-inventory/` holds stock health and restock planning surfaces.
- `admin-users/` holds member management surfaces.
- `admin-overview/` is the earlier admin scaffold and can be retired after the new modules fully replace it.
